Loop Engineering from Scratch

Teach AI to finish tasks correctly and then stop on its own. 10 lessons, Python runs directly in the browser, exercises auto-graded online, zero install.

Prompt engineering is "craft one message, see one result"; loop engineering is "design a system that act -> verify -> repeat on its own until the goal is met or a limit is hit." Read the chapter on the left, run demos and write exercises on the right -- all autograder checks green means you pass (progress saved in your browser).

After all ten lessons pass: the Capstone final assessment requires a local environment (subprocess/cron). Please clone the GitHub repo, follow capstone/SPEC.md, and use python3 progress.py to see full progress.
